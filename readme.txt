Skypro-Kanban - это инструмент управления задачами, который помогает пользователю 
в их наглядном представлении.

Авторизованному пользователю предоставлено цифровое пространство, в котором можно 
создать и визуально упорядочить пользовательские задачи.

Как это работает:

Для добавления задачи в правом верхнем углу приложения представлен функционал "Создать новую задачу".

После добавления новой задачи она отображается на доске в соответствии параметрами, 
выбранными при её создании.

С помощью карточек задач и столбцов на доске Kanban предполагается разделить пользовательские 
задачи на категории и статус выполнения.

При этом у пользователя сохраняется возможность редактирования задачи путём вызова соответствующего
окна (три точки в правом верхнем углу каждой карточки с задачей).

Под редактированием понимается изменение описания, сроков исполнения, а также статуса 
и категории задачи.

В том же окне расположен функционал удаления задачи.

Перечень всех задач конкретного пользователя, а также данные отдельно каждой задачи 
хранятся удалённо и, с целью отображения актуальных данных, при совершении пользователем 
действий, связанных с манипуляцией Kanban доской, производится запрос требуемой информации, 
в процессе чего отображается анимированная загрузка, информирующая пользователя 
о соответствующем этапе работы приложения.

Для улучшения пользовательского опыта была добавлена возможность изменения темы всего 
приложения. Информация о выборе той или иной темы сохраняется - таким образом 
при последующем посещении пользователь увидит ту тему, которая была выбрана им ранее. 
Данный функционал доступен в меню пользователя (надпись введённого при авторизации 
логина в правом верхнем углу приложения).
